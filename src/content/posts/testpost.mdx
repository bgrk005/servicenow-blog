# ServiceNow: Safely Querying GlideRecord (with Examples)

Learn how to safely query data using the GlideRecord API in ServiceNow.

```js
var gr = new GlideRecord("incident");
gr.addQuery("active", true);
gr.query();
while (gr.next()) {
  gs.info(gr.number + ": " + gr.short_description);
}
```
