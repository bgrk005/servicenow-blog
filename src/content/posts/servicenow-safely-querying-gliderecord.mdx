When working with ServiceNow’s GlideRecord API, it’s important to structure queries
so they are safe, performant, and readable.

```js
var gr = new GlideRecord("incident");
gr.addQuery("active", true);
gr.query();
while (gr.next()) {
  gs.info(gr.number + ": " + gr.short_description);
}
